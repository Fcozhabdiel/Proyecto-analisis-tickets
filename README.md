# Sistema de Análisis de Tickets de Soporte TI

##Descripción
Este proyecto tiene como objetivo el desarrollo de una aplicación web que sea funcional para el análisis y visualización de tickets de para el área de TI de Grupo SC Automotriz.

La solución busca mejorar la toma de decisiones mediante el uso de indicadores de desempeño como KPIs, así como optimizar los tiempos de respuesta e identificar problemas recurrentes. 

##Objetivo
Desarrollar una app web que permita consultar, filtrar y analizar la información de tickets de soporte técnico.

##Tecnologías propuestas
- Node.js
- MySQL
- HTML
- CSS
- JavaScript

##Funcionalidades principales
- Dashboard con indicadores
- Consulta y filtrado de tickets
- Generación de reportes
- Identificación de patrones en incidencias
