# Sistema de Análisis de Tickets de Soporte TI

##Descripción
Este proyecto tiene como objetivo el desarrollo de una aplicación web para el análisis y visualización de tickets de soporte técnico en el área de Tecnologías de la Información de Grupo SC Automotriz.

La solución busca mejorar la toma de decisiones mediante el uso de indicadores clave de desempeño como KPIs, así como optimizar los tiempos de respuesta e identificar problemas recurrentes dentro del proceso de atención de incidencias.

##Objetivo
Desarrollar una plataforma web que permita consultar, filtrar y analizar la información de tickets de soporte técnico, contribuyendo a una gestión más eficiente del área de TI.

##Tecnologías propuestas
- Node.js
- MySQL
- HTML
- CSS
- JavaScript

##Funcionalidades principales
- Dashboard con indicadores
- Consulta y filtrado de tickets
- Generación de reportes
- Identificación de patrones en incidencias
