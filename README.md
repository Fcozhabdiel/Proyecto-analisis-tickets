# Proyecto-analisis-tickets
Proyecto de Desarrollo Tecnológico

Sistema de Análisis de tickets de Soporte TI

#El objetivo de este proyecto es el desarrollo de una app web para el análisis de tickets para el area de TI de Grupo SC Automotriz.

#Se desarrollara una app web que permita consultar, filtrar y analizar la informacion de tickets enviados al departamento de soporte tecnico. 

#Tecnologias
- Node.js
- MySQL
- HTML
- CSS
- JavaScript

#Funciones del sistema
- Dashboard con indicadores
- Consulta y filtrado de tickets
- Generacion de reportes
- Identificar patrones en incidencias
